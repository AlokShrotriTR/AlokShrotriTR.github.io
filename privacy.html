ACTR Privacy Policy

Information We Collect

ACTR collects minimal information necessary to provide its services:

  • User identity information from Microsoft Teams
  • Incident numbers entered by users
  • Meeting details created through the app

How We Use Information

Information is used solely to:

  • Retrieve incident details from ServiceNow
  • Create Teams meetings for incident resolution
  • Update ServiceNow incidents with meeting information

Data Storage

ACTR does not store personal data. All information is processed in real-time and passed securely to Microsoft Graph API and ServiceNow API.

Third-Party Services

ACTR integrates with:

  • Microsoft Graph API (for Teams meetings)
  • ServiceNow API (for incident management)

Contact

For privacy concerns, contact your organization's IT administrator.
